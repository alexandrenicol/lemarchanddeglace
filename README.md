## requirements
- node + npm

## install + start
- git clone https://github.com/alexandrenicol/lemarchanddeglace.git
- cd lemarchanddeglace
- npm install
- npm run start (or npm run build)

## Stack:
- React + Redux + Sass
- CD: GitHub -> AWS CodePipeline -> S3 -> CloudFront
